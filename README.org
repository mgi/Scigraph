* Scigraph

A try to reimplement Scigraph (see [fn:1]) on top of Robert Strandh's
CLIMatis (see [fn:2])

Not much here yet, you can try this:

(ql:quickload 'clim3-scigraph)
(scigraph:run)
#+END_SRC
